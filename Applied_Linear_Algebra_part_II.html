Nonsingular Matrix¶

Theorem: a matrix $A \in \mathbb{R}^{m×n}$ with $m \ge n$ has full rank if and only if it maps no two distinct vectors to the same vector, and $Ax = Ay \Rightarrow x = y$.¶

Singular matrix : a square matrix $A \in \mathbb{R}^{n×n}$ that is not full rank.¶

Nonsingular matrix : a square matrix $A \in \mathbb{R}^{n×n}$ of full rank.¶

If $A$ is nonsingular, we can uniquely express any vector $y \in \mathbb{R}^n$ as $y=Ax$ for some unique $x \in \mathbb{R}^n$. If we chose vectors $e_i =Ax_i, i = 1,...,n$, we can write¶

$AX=A[x_1 ··· x_n]=[Ax_1 ··· Ax_n]=[e_1 ··· e_n]=I$¶

Matrix Inverse¶

Matrix inverse: the inverse of a square matrix $A \in \mathbb{R}^{n×n}$ is the matrix $B \in \mathbb{R}^{n×n}$ such that¶

$AB = BA = I$ where $I$ is the identity matrix¶

The inverse of $A$ is denoted by $A^{−1}$.¶

Any square, nonsingular matrix $A$ has a unique inverse $A^{−1}$ satisfying $AA^{−1} = A^{−1}A = I$.¶

Invertible Matrix Theorem¶

Let $A \in \mathbb{R}^{n×n}$ be a square matrix, then the following statements are equivalent:¶

  • $A$ is invertible, $A$ is nonsingular.
  • There exists a matrix $A^{1}$ such that $AA^{1} = I_n = A^{1}A$.
  • $Ax = b$ has exactly one solution for each $b \in \mathbb{R}^n$.
  • $Az=0$ has only the trivial solution $z=0$,i.e. $N(A)=$
  • The columns of $A$ are linearly independent.
  • $A$ is full rank, i.e. rank($A$) = $n$.
  • $det(A)\ne 0$.
  • 0 is not an eigenvalue of $A$.

One of the fundamental problems in linear algebra:¶

Given $A\in \mathbb{R}^{m×n}$ and $b \in \mathbb{R}^m$, find $x \in \mathbb{R}^n$ such that $Ax = b$.¶

The set of solutions of a linear system can behave in three ways:¶

  1. The system has one unique solution.
  2. The system has infinitely many solutions.
  3. The system has no solution.

For a solution $x$ to exist, we must have $b \in R(A)$.¶

Where R( ) is the row space, m is the row dim, and n is the column dim¶

  • If $b \in R(A)$ and rank($A$) = n, then there is a unique solution (usually square systems: $m = n$).
  • If $b \in R(A)$ and rank($A$) $\lt n (N (A) \ne )$, then there are infinitely many solutions (usually underdetermined systems: $m < n$).
  • If $b \notin R(A)$, then there is no solution (usually overdetermined systems: $m > n$).

Gaussian Elimination (row reduction)¶

How do we solve $Ax = b$, $A \in \mathbb{R}^{nn}$? For some types of matrices, $Ax = b$ is easy to solve.

  • Diagonal matrices: equations independent, so $x_i = b_i/a_{ii}$, $\forall i$
  • Upper/Lower triangular matrices: back/forward substitution.

Gaussian Elimination: Transform the system $Ax = b$ into an upper triangular system $Ux = y$ using elementary row operations. The following are elementary row operations that can be applied to the augmented system $[A | b]$ to introduce zeros below the diagonal without changing the solution set x:

  1. Multiply row by a scalar.
  2. Add scalar multiples of one row to another.
  3. Permute rows.

Gaussian Elimination and LU Decomposition¶

We can think of the elementary row operations applied to A as linear transformations,¶

which can be represented as a series of lower triangular matrices, $L_1, · · · , L_{n−1}$.¶

$L_{n−1} ···L_1A = U$ or $L^{−1}A = U, L^{−1} = L^{−1}_1 ··· L^{−1}_{n−1}$¶

rearranging yields $A = LU$.¶

If we can factor $A$ as $A = LU$, then we can solve the system¶

$Ax = b$ by solving two triangular systems:¶

  • Solve $Ly = b$ using forward substitution,
  • Solve $Ux = y$ using backward substitution.

Gaussian elimination implicitly computes the LU factorization.¶

Partial Pivoting¶

Pure Gaussian elimination can not be used to solve general linear systems because we may encounter a zero on the diagonal during the process, e.g.¶

$A = \begin{bmatrix} 0 & 1 \¶

                     1 & 2
     \end{bmatrix}                
                     $

In practical implementations of the algorithm, the rows of the matrix must be permuted to avoid division by zero (see examples with and without permutation above).¶

Partial pivoting swaps rows if an entry below the diagonal of the current column is larger in absolute value than the current diagonal entry (pivot element).¶

Permutations¶

The reordering, or permutation, of the columns in partial pivoting can be represented as a linear transformation.¶

Permutation matrix : a square, binary matrix $P \in \mathbb{R}^{n×n}$ that has exactly one entry 1 in each row and each column.¶

Left-multiplication of a matrix $A$ by a permuation matrix reorders the rows of $A$, while right-multiplication reorders the columns of $A$.¶

For example, let $P = \begin{bmatrix} 0 & 1 & 0 \\ 0 & 0 & 1 \\ 1 & 0 & 0 \end{bmatrix}$, and $A = \begin{bmatrix} 1 & 2 & 3 \\ 10 & 20 & 30 \\ 100 & 200 & 300 \end{bmatrix}$,

then the row and column permutations are, respectively:¶

$PA = \begin{bmatrix} 10 & 20 & 30 \\ 100 & 200 & 300 \\ 1 & 2 & 3 \end{bmatrix}$, and $AP = \begin{bmatrix} 3 & 2 & 1 \\ 30 & 20 & 10 \\ 300 & 200 & 100 \end{bmatrix}$

LU Decomposition¶

Let $A \in \mathbb{R}^{n×n}$ be a square matrix, then the LU decomposition factors $A$ into the product of a lower triangular matrix $L \in \mathbb{R}^{n×n}$ and an upper triangular matrix $U \in \mathbb{R}^{n×n} A = LU$.¶

When partial pivoting is used to permute the rows, we obtain a decomposition for the form $L_{n−1}P_{n−1} · · · L_1P_1A = U$, which can be written in form: $L^{−1}PA = U$.¶

In general, any square matrix $A \in \mathbb{R}^{n×n}$ (singular or nonsingular) has a factorization¶

$P A = LU$, where $P$ is a permutation matrix.¶

Cholesky and Positive Definiteness¶

Positive definite matrix: a matrix $A \in \mathbb{R}^{n×n}$ is positive definite if $z^TAz \gt 0$ for every $z \in \mathbb{R}^n$, often denoted by $A\succ 0$.¶

  • Positive semi-definite matrix: $z^T Az \ge 0$ for every $z \in \mathbb{R}^n$.
  • Negative definite matrix: $z^T Az \lt 0$ for every $z \in \mathbb{R}^n$.

A symmetric matrix $A$ is positive definite ($A \succ 0$) if and only if there exists a unique lower triangular matrix $L$ such that $A = LL^T$ .¶

This factorization is called the Cholesky decomposition of $A$.¶

The Cholesky algorithm, a modified version of Gauss elimination, is used to compute the factor $L$.¶

Basic idea: In Gauss-elimination we use row operations to introduce zeros below the diagonal in each column. To maintain symmetry, we can apply the same operations to the columns of the matrix to introduce zeros in the first row.¶

Orthogonality¶

Recall that two vectors $x, y \in \mathbb{R}^n$ are orthogonal if $x^T y = 0$ (their dot product is zero), and a set of vectors {$u_1, \dots , u_n$} is orthonormal if¶

$u^T_iu_j= \begin{cases}¶

1, & \text{for } i=j\\
0, & \text{for } i\ne j

\end{cases}$, $|u_i|=1,\forall i$

Orthogonal (Unitary) Matrix : a matrix $Q \in \mathbb{R}^{n×n}$ is orthogonal if its columns $q_1, \dots , q_n \in \mathbb{R}^n$ form an orthonormal set:¶

$QQ^T = \begin{bmatrix} q_1^T \\ \dots \\ q_n^T \end{bmatrix} \begin{bmatrix} q_1 & \dots & q_n \end{bmatrix} = I_n$¶

If $Q$ is orthogonal, then $Q^{−1} = Q^T$ (inverse is easy to compute!)¶

Orthogonal transformations preserve lengths and angles and represent rotations or reflections.¶
